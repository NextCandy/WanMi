# fix: 恢复精简产品边界并优化首页性能

## 变更

- 合并 `main` 中已经完成的注册商账户、DNS、求购线索删除，修复落后分支误发布。
- 恢复被错误编码破坏的 UTF-8 中文源码。
- 搜索按钮固定为右侧 72px；域名卡片只保留收藏、复制、速览三枚纯图标，删除“我想要”。

- 新增 `0016`，兼容线上旧 `registrar` 与全新安装 `registrar_name`，删表同时保护域名、分类和通知历史。
- README、部署说明、设计规范和 HANDOFF 改为当前七模块产品边界。
